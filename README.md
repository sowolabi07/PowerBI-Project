# 📊 Retail Sales Performance Dashboard (Power BI)

## 📌 Project Overview

This project analyzes retail sales data to uncover insights into overall business performance, product profitability, and customer behavior.
The dashboard is designed to support decision-making through interactive visuals and advanced analytical techniques.

---

## 🎯 Objectives

* Analyze overall sales and profit performance
* Identify high-performing and underperforming products
* Detect loss-making products despite high sales
* Understand customer segments and profitability
* Enable interactive exploration using slicers

---

## 🔍 Key Insights

* Strong overall revenue growth across regions
* Technology category leads in both sales and profitability
* Several **high-revenue products are unprofitable**, indicating pricing/discount issues
* Customer analysis shows **margin leakage in certain segments**
* Revenue is concentrated among a few top products and customers

---

## 🧠 Key Features

* Interactive slicers (Category, Segment, Region, Date)
* Drill-down and hover-based insights (tooltips)
* Profitability matrix using scatter chart with break-even analysis
* Clean and consistent dashboard design

---

## 🛠 Tools & Skills Used

* Power BI
* DAX (CALCULATE, DIVIDE, Time Intelligence)
* Data Modeling
* Data Visualization & Storytelling

---

## 🚀 How to Use

1. Open the `.pbix` file in Power BI Desktop
2. Use slicers to filter by Category, Segment, or Region
3. Hover over visuals to explore detailed insights
4. Navigate across pages for different perspectives
